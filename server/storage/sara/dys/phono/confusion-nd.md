## Description du thread

Les sons **[n]** et **[d]** se distinguent par **nasalité (nez vs bouche)**. Les deux : pointe de la langue contre les dents. Pour [n], l'air sort par le nez. Pour [d], par la bouche.

Tu vas travailler 5 objectifs : sentir la différence en bouche, l'entendre, la lire, l'écrire dans un mot, puis dans une phrase.
