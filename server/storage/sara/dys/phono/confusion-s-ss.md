## Description du thread

Les sons **[s]** et **[z]** se distinguent par **position dans le mot**. Le S se prononce différemment selon sa position. Entre 2 voyelles, S = [z] (rose). Au début ou doublé (SS), S = [s] (soleil, poisson).

Tu vas travailler 5 objectifs : sentir la différence en bouche, l'entendre, la lire, l'écrire dans un mot, puis dans une phrase.
