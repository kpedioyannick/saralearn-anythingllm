## Description du thread

Les sons **[k]** et **[s]** se distinguent par **règle d'orthographe**. Le C se prononce différemment selon la voyelle qui suit. Pour [k], C devant a, o, u (cadeau, coq, cube). Pour [s], C devant e, i (cerise, cinéma) OU Ç devant a, o, u (ça, garçon).

Tu vas travailler 5 objectifs : sentir la différence en bouche, l'entendre, la lire, l'écrire dans un mot, puis dans une phrase.
